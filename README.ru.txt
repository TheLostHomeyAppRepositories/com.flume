Приложение Flume для Homey с легкостью интегрирует ваш умный счетчик воды Flume с системой умного дома Homey, предоставляя мощные возможности для контроля за использованием воды. Это приложение предоставляет данные о потреблении воды в вашем доме в реальном времени, помогая экономить воду, обнаруживать утечки и снижать счета за коммунальные услуги.

С помощью приложения Flume для Homey вы можете настроить интеллектуальную автоматизацию для оптимизации управления водой. Получайте мгновенные уведомления о необычных паттернах использования воды, автоматизируйте устройства, экономящие воду, в периоды пикового использования, или интегрируйте с другими компонентами умного дома для улучшения усилий по экономии воды в вашем доме.

Независимо от того, стремитесь ли вы уменьшить свой экологический след, предотвратить дорогостоящие повреждения от воды или расширить возможности вашего умного дома, приложение Flume для Homey предлагает умное управление водоснабжением прямо под рукой.

Для использования этого приложения вам потребуется активный аккаунт Flume.

Запросить персональный клиентский ключ и секрет для доступа к API можно через портал Flume.
