La aplicación Flume Homey integra sin esfuerzo tu monitor de agua inteligente Flume con el sistema de hogar inteligente Homey, ofreciendo potentes conocimientos y control sobre tu uso del agua. Esta app proporciona datos en tiempo real sobre el consumo de agua de tu hogar, ayudándote a conservar agua, detectar fugas y reducir las facturas de servicios con facilidad.

Con la aplicación Flume Homey, puedes establecer automatizaciones inteligentes para optimizar la gestión del agua. Recibe alertas instantáneas por patrones inusuales de uso del agua, automatiza dispositivos de ahorro de agua durante los momentos de mayor consumo, o intégralo con otros componentes del hogar inteligente para mejorar los esfuerzos de conservación del agua en toda tu casa.

Ya sea que estés comprometido con reducir tu huella ambiental, prevenir daños costosos por agua, o expandir las capacidades de tu hogar inteligente, la aplicación Flume Homey ofrece gestión inteligente del agua directamente en tus manos.

Para utilizar esta app, necesitas una cuenta activa de Flume.

Solicitar una clave personal de cliente y un secreto para acceder a la API se puede hacer a través del portal de Flume.
