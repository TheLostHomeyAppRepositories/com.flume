Flume Homey-appen integrerar enkelt din Flume smarta vattenmätare med Homeys smarta hemsystem, vilket ger kraftfulla insikter och kontroll över din vattenanvändning. Den här appen ger dig realtidsdata om hushållets vattenförbrukning, vilket hjälper dig att spara vatten, upptäcka läckor och minska dina vattenräkningar enkelt.

Med Flume Homey-appen kan du ställa in smarta automatiseringar för att optimera vattenhanteringen. Få omedelbara varningar för ovanliga vattenanvändningsmönster, automatisera vattenbesparande enheter under högförbrukningstider, eller integrera med andra smarta hemanordningar för att öka vattenbevarande insatser i hela ditt hem.

Oavsett om du är engagerad i att minska din miljöpåverkan, förhindra kostsam vattenskador, eller utöka dina smarta hemfunktioner, levererar Flume Homey-appen smart vattenhantering direkt till dina fingertoppar.

För att använda den här appen behöver du ett aktivt Flume-konto.

Att begära en personlig klientnyckel och hemlighet för att få tillgång till API:et kan göras via Flume-portalen.
